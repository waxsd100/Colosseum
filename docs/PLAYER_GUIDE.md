# Colosseum プレイヤーガイド

色付きカーペットをカジノチップとして使う、闘技場ベッティングの遊び方。
金額・秒数・還元率などの数値はサーバー設定により変わる場合がある（本ガイドは標準設定の値）。

---

## 試合の流れ

1. **募集開始** — 闘技者はチーム待機エリアに入る
2. **参加締切** — ベット受付開始。観客はチップを買ってベット
3. **ベット締切** — 試合開始
4. **勝敗確定** — 的中者に配当、外れは没収

---

## チップ

| 色 | 額面 | | 色 | 額面 |
|---|---:|---|---|---:|
| 茶 | 1 E | | 黄 | 5,000 E |
| 紫 | 5 E | | 橙 | 10,000 E |
| 青 | 10 E | | 桃 | 50,000 E |
| 水 | 50 E | | 赤 | 100,000 E |
| 浅葱 | 100 E | | 白 | 500,000 E |
| 緑 | 500 E | | 黒 | 1,000,000 E |
| 黄緑 | 1,000 E | | | |

| コマンド | 説明 |
|---|---|
| `/chip <金額>` | 金額分を自動分割で購入 |
| `/chip <額面> <枚数>` | 指定額面を指定枚数購入 |
| `/chip info` | 額面と色の対応表 |
| `/chip balance` | 手持ちチップの内訳と合計 |
| `/chip cashout` | 手持ちチップを全額換金 |

- 購入できるのはベット受付中（またはカジノイベント中）のみ。`info` / `balance` / `cashout` は常時使用可。
- 購入時にアドベンチャーモードへ切り替わり、回収用ハサミが配布される。換金時に元へ戻る。
- 未使用チップは**ベット締切時・試合終了時・ログアウト時に自動換金**される。持ち越しはできない。

## ベット方法（観客）

### 方法1: チップを置く

対象チームのベットエリア（チームカラーのパーティクル枠内）にチップを設置する。設置した額面がベットとして加算される。

- 締切前なら、ハサミで自分のカーペットを壊して取消できる（他人の分は不可）。

### 方法2: コマンド

| コマンド | 説明 |
|---|---|
| `/bet <チーム> <金額>` | 所持金から直接ベット。**取消不可** |
| `/bet odds` | 現在のオッズとプール総額 |
| `/bet info` | 自分のベット状況と予想配当 |

### 共通ルール

- 複数チームへの同時ベット可。
- 闘技者はベット不可。
- 締切30秒前からオッズ非公開（ブラインド）。
- ベット後にログアウトしても有効。結果は次回ログイン時に自動精算・通知される。

## オッズと配当

パリミュチュエル方式。全員のベットをプールし、天引き後に勝ちチームへ賭けた人で分配する。

```
ベット総額（プール）
  ├─ 勝者闘技者への還元 12%
  ├─ 運営手数料         2%
  └─ 観客配当プール    86%
```

**オッズ = 観客配当プール ÷ 勝ちチームへのベット総額**（締切まで変動、30秒ごとに表示）

例: 総プール 100,000 E、勝ちチームへのベットが 20,000 E の場合、オッズは 4.3 倍。10,000 E のベットなら配当 43,000 E。

- 外れたベットは全額没収。
- オッズが1.0倍を下回る場合は元返し（的中者にはベット額をそのまま払い戻す）。表示は「1.00倍(元返し)」。
- 引き分け・試合中止の場合は全額返金。

---

## ジャックポット

運営手数料や配当の端数が毎試合積み立てられる。**勝利チームへのベット比率が30%未満**（大穴）のとき発動し、積立全額が勝ちベッターの配当に上乗せされる。積立額は参加締切時にアナウンスされる。

---

## ダブルアップ

配当が 100 E 以上のとき、受取時に選択肢が表示される。

| 選択 | 内容 |
|---|---|
| DOUBLE UP | 配当を受け取らず、次の試合に全額自動ベット |
| CASH OUT | その場で受け取る（15秒未選択の場合はこちらに自動確定） |

連勝ボーナス倍率: 2連勝 ×1.2 / 3連勝 ×1.5 / 4連勝 ×2.0 / 5連勝以上 ×3.0

- 自動ベット先は前回と同じチーム名（存在しない場合は先頭のチーム）。
- 次の試合で負けると保留額は全額没収。

## 闘技者として参加する

- 募集中にチーム待機エリアに入ると参加、出ると取消。コマンド不要。
- 参加締切時に確定し、試合開始でアリーナへテレポートされる。
- 参加費が設定されている場合、不足時は「借金して参加 / 辞退」を選択する（30秒無回答は辞退扱い）。

### 報酬

| 報酬 | 内容 |
|---|---|
| 最低保証金 | 勝敗に関係なく全闘技者に 100 E |
| 勝者還元 | ベット総額の 12% を勝利チームで分配 |
| 参加費払い戻し | 勝利チームのみ全額返還 |

### ルール

- チーム全滅方式。死亡またはログアウトで脱落。
- 戦闘エリア（赤いパーティクル境界）の外に出ると警告が表示され、10秒以内に戻らないと毎秒最大HPの10%のダメージを受ける。
- 全チーム同時全滅などの場合は引き分け（ベット・参加費とも全額返金）。

## デスマッチ

闘技者同士が賭け金を出し合うモード。ベット受付中に闘技者が提案できる。

| コマンド | 説明 |
|---|---|
| `/arena deathmatch <総額>` | 提案（総額を人数割りした額が1人あたりの賭け金） |
| `/arena deathmatch all` | ALL-IN（全財産）で提案 |
| `/arena deathmatch yes` / `no` | 賛成 / 反対（チャットのボタンでも可） |
| `/arena deathmatch info` | 投票状況を確認 |
| `/arena deathmatch cancel` | 提案を取り下げ（提案者のみ） |

### 成立条件

- 投票時間は20秒。提案者は自動的に賛成扱い。
- チーム5人以下は全員一致、6人以上は過半数の賛成が必要。
- 全チームの承認で成立。1チームでも否決すれば不成立。
- 提案は1試合につき2回まで。

### 成立後

- 試合開始時に1人あたりの賭け金が徴収される（不足分は借金）。
- 勝利チームが賭け金プールを山分けする。
- ALL-IN はチーム合計所持金の多い方に金額を合わせるため、少ない側は差額を借金として負担する。

## 補足

- 他人の設置チップは壊せない。回収できるのは自分の分だけ、かつベット受付中のみ。
- 試合開始後のカーペット回収は不可。
- インベントリに入りきらないチップは足元にドロップする。
- オッズが「???」表示のときはブラインド中。
