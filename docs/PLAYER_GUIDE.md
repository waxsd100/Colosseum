# 🏟️ コロシアム 遊び方ガイド（プレイヤー向け）

このサーバーでは、**カーペットをカジノチップに見立てて**闘技場の試合にベットしたり、自分が闘技者として戦って賞金を稼いだりできます。
このガイドでは、一般プレイヤーができることをすべて説明します。

> 💡 文中の金額・秒数・還元率などはサーバー設定によって変わることがあります（本ガイドは標準設定の値です）。

---

## 🎮 遊びの流れ（ざっくり）

```
① 募集開始 ──「⚔ 闘技者募集中！」のアナウンス
      │        戦いたい人 → チーム待機エリアに入るだけで参加
      ▼
② 参加締切 ──「🔒 参加者締切 & ベット受付開始！」
      │        観客 → チップを買ってベット
      ▼
③ ベット締切 ── 試合開始！
      ▼
④ 勝敗決定 ── 的中した人に配当、外れた人は没収
```

観客として**ベットで儲ける**か、闘技者として**戦って稼ぐ**か、好きな方で参加できます。

## 💰 チップの基本

チップの実体は**色付きカーペット**です。色ごとに額面が決まっています。

| 色 | 額面 | | 色 | 額面 |
|---|---:|---|---|---:|
| 茶 | 1 E | | 黄 | 5,000 E |
| 紫 | 5 E | | 橙 | 10,000 E |
| 青 | 10 E | | 桃 | 50,000 E |
| 水 | 50 E | | 赤 | 100,000 E |
| 浅葱（マゼンタ） | 100 E | | 白 | 500,000 E |
| 緑 | 500 E | | 黒 | 1,000,000 E |
| 黄緑 | 1,000 E | | | |

### チップのコマンド

| コマンド | 説明 |
|---|---|
| `/chip <金額>` | 金額分のチップを自動でいい感じの組み合わせで購入 |
| `/chip <額面> <枚数>` | 指定額面のチップを指定枚数購入（例: `/chip 1000 5`） |
| `/chip info` | 額面と色の対応表を表示 |
| `/chip balance` | 手持ちチップの内訳と合計を表示 |
| `/chip cashout` | 手持ちチップをすべて換金して所持金に戻す |

**知っておくこと:**

- チップを**購入できるのはベット受付中（またはカジノイベント中）だけ**です。`info` / `balance` / `cashout` はいつでも使えます。
- チップを買うと自動で**アドベンチャーモード**になり、チップ回収用の**「チップ回収ハサミ」**が配られます。換金すると元のゲームモードに戻ります。
- **チップの持ち逃げはできません。** 以下のタイミングで自動的に換金されます:
  - ベット締切時（手持ちの未使用チップ）
  - 試合終了時
  - サーバーからログアウトした時

## 🎲 ベットのやり方（観客）

ベット受付中（参加者締切〜ベット締切の間）に、2つの方法でベットできます。

### 方法① チップを置く（おすすめ・雰囲気重視）

1. `/chip <金額>` でチップを購入
2. 応援したいチームの**ベットエリア**（チームカラーのパーティクルで縁取られた場所）に行く
3. チップ（カーペット）を**エリア内に設置**する → その額面ぶんベット完了！

- 置くたびに合計ベット額に加算されます。
- **気が変わったら**: 締切前なら、ハサミで**自分の置いたカーペットを壊せば返却**されます（他人のチップは壊せません）。

### 方法② コマンドでベット（手軽・スピード重視）

| コマンド | 説明 |
|---|---|
| `/bet <チーム名> <金額>` | 所持金から直接ベット（例: `/bet Warriors 5000`） |
| `/bet odds` | 現在のオッズとプール総額を表示 |
| `/bet info` | 自分のベット状況と予想配当を表示 |

> ⚠ **コマンドでのベットは取消できません。** 取り消せるのはカーペット設置ぶんだけです。

### ベットのルール

- **複数のチームに同時にベットできます**（ヘッジベット）。結果発表時に差し引きが表示されます。
- **闘技者はベットできません**（自分の試合には賭けられない）。
- **ブラインドタイム**: 締切の30秒前からはオッズ・プール額が**非公開**になります。駆け込みベットは自己責任で！
- ベット後にログアウトしても**ベットは有効**です（結果は次回ログイン時に精算されます）。

## 📊 オッズと配当の仕組み

このコロシアムは**パリミュチュエル方式**（公営競技と同じ方式）です。胴元と戦うのではなく、**外れた人のベットを当てた人で山分け**します。

```
全員のベット総額（プール）
  ├─ 勝った闘技者への還元 …… 12%
  ├─ 運営手数料 ……………………  2%
  └─ 観客配当プール …………… 86% ← 勝ちチームに賭けた人で山分け！
```

**オッズ = 観客配当プール ÷ 勝ちチームへのベット総額**

つまり、**人気のないチーム（賭ける人が少ないチーム）ほどオッズが高く**、当たれば大きく儲かります。オッズは締切まで変動し続けます（30秒ごとにチャットに流れます）。

**例:** 総プール 100,000 E、Warriors に 20,000 E しか賭けられていないとき
→ 観客配当プールは 86,000 E → Warriors のオッズは **4.3倍**
→ あなたが Warriors に 10,000 E 賭けて勝てば **43,000 E** もらえます 🎉

- 外れたベットは**全額没収**です。
- 引き分けや試合中止のときは**全額返金**されます。

## 🎰 ジャックポット

運営手数料や配当の端数などが**ジャックポット**として毎試合積み立てられていきます。

- **勝利チームへのベット割合が全体の30%未満**だったとき（＝大穴が勝ったとき）に**発動**！
- 発動すると、積立金の**全額**が勝ちベッターの配当に上乗せされます。花火と共に🎆
- 現在の積立額は参加者締切時のアナウンスで確認できます。

大穴狙いにはロマンがあります。

## ⚡ ダブルアップ（連勝チャレンジ）

ベットに勝つと、配当受取時に**2つのボタン**が表示されます（配当100 E以上のとき）。

| 選択肢 | 内容 |
|---|---|
| **▲ DOUBLE UP** | 配当を受け取らず、**次の試合に全額自動ベット**する |
| **▼ CASH OUT** | 配当をその場で受け取る（15秒間未選択の場合も自動でこちら） |

DOUBLE UP を続けて勝つほど**連勝ボーナス倍率**が上がります:

| 連勝数 | ボーナス倍率 |
|---:|---:|
| 2連勝 | ×1.2 |
| 3連勝 | ×1.5 |
| 4連勝 | ×2.0 |
| 5連勝以上 | ×3.0 |

- 自動ベット先は**前回と同じチーム名**です（次の試合にそのチームがなければ先頭のチーム）。
- ⚠ **次の試合で負けると保留額は全額没収**です。引き際が肝心！

## ⚔️ 闘技者として参加する

### 参加方法

**「⚔ 闘技者募集中！」のアナウンスが出たら、チームの待機エリアに入るだけ。** コマンド不要です。エリアから出れば参加取消になります。参加締切時に待機エリアにいた人が闘技者として確定し、試合開始と同時にアリーナへテレポートされます。

### 参加費について

サーバー設定で参加費がある場合、試合開始時に徴収されます。所持金が足りないときは **[借金して参加] / [辞退する]** の選択ボタンが表示されます（30秒以内に未回答だと自動辞退）。

### 闘技者の報酬

戦うだけでお金がもらえます:

| 報酬 | 内容 |
|---|---|
| 🪙 最低保証金 | 勝敗に関係なく全闘技者に **100 E** |
| 🏆 勝者還元 | ベット総額の **12%** を勝利チームで山分け |
| 🔄 参加費払い戻し | 勝利チームは参加費が全額戻ってくる |

観客のベットが盛り上がるほど、闘技者の取り分も増えます。

### 試合のルール

- 基本は**チーム全滅方式**: 相手チームを全員倒せば勝ち。**死亡またはログアウトで脱落**です。
- **戦闘エリアから出てはいけません。** 赤いパーティクルの境界線の外に出ると警告が表示され、**10秒以内に戻らないと毎秒最大HPの10%のダメージ**を受け続けます。
- 相手がモンスターの試合もあります（モンスター全滅で勝利）。
- 両チーム同時全滅などの場合は**引き分け**となり、ベット・参加費はすべて返金されます。

## 💀 デスマッチ（闘技者専用・高額賭け試合）

闘技者同士が**自分たちのお金を賭け合う**真剣勝負モードです。ベット受付中に闘技者が提案できます。

| コマンド | 説明 |
|---|---|
| `/arena deathmatch <総額>` | デスマッチを提案（総額を人数割りした額が1人あたりの賭け金） |
| `/arena deathmatch all` | **ALL-IN** デスマッチを提案（全財産を賭ける！） |
| `/arena deathmatch yes` / `no` | 提案に賛成 / 反対（チャットのボタンでもOK） |
| `/arena deathmatch info` | 投票状況を確認 |
| `/arena deathmatch cancel` | 提案を取り下げ（提案者のみ） |

### 成立条件（投票）

- 投票時間は**20秒**。提案者は自動的に賛成扱いです。
- **5人以下のチーム → 全員一致**、**6人以上のチーム → 過半数**の賛成が必要です。
- **すべてのチーム**が承認して初めて成立します。1チームでも否決したら不成立。
- 提案できるのは1試合につき**2回まで**。

### 成立したら

- 試合開始時に1人あたりの賭け金が徴収されます（足りない分は**借金**になります）。
- 勝利チームが**賭け金プールを山分け**します。
- **ALL-IN** の場合: チーム合計所持金が多い方に金額を合わせるため、少ない側は不足分を借金として背負います。負けたら文字通りすべてを失います。覚悟のある者だけどうぞ。

## ❓ よくある質問

**Q. ベットしたあとログアウトしてしまった！**
A. ベットは有効なままです。試合結果は保存され、**次回ログイン時に配当の受け取り（または没収の通知）が自動で行われます。**

**Q. チップを持ったまま試合が終わったら？**
A. 使わなかったチップはベット締切時・試合終了時に自動で換金されて所持金に戻ります。損はしません。

**Q. 他人の置いたチップを壊して奪える？**
A. できません。回収できるのは自分のチップだけ、しかもベット受付中だけです。

**Q. インベントリがいっぱいでチップが入らない！**
A. 入りきらない分は足元にドロップします。拾い忘れに注意してください。

**Q. 間違えたチームにベットしてしまった！**
A. カーペット設置ぶんは締切前ならハサミで壊して回収→置き直しできます。`/bet` コマンドのベットは取消できないので慎重に。

**Q. オッズが「???」になってる！**
A. 締切直前のブラインドタイムです。最終オッズは締切後のお楽しみ。
